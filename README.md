# Лабораторная работа №14-16 - Интернет-магазин "TechStore"

**ФИО:** Мефедова Алина Михайловна

**Группа:** ИСП-232

**Дата:** 06.03.2026
## Описание проекта

Многостраничный сайт интернет-магазина электроники "TechStore" с адаптивной вёрсткой. В проекте используются смайлики вместо изображений для упрощения и ускорения разработки.

## Реализованные страницы

- **Главная** — приветственный баннер, популярные товары со скидками, преимущества
- **Каталог** — сетка из 9 карточек товаров с фильтрами и скидками
- **О нас** — информация о магазине, команда (с использованием смайликов)

## Реализованные функции

- Адаптивное навигационное меню
- Карточки товаров с hover-эффектами
- Скидки на товары (наушники -20%, часы -13%)
- Зачеркнутые старые цены и выделенные новые
- Анимированные бейджи со скидками
- CSS Grid для каталога (3 колонки на desktop, 2 на tablet, 1 на mobile)
- Flexbox для навигации и футера
- Адаптивная вёрстка (desktop/tablet/mobile)
- Единая цветовая схема и типографика
- Семантическая HTML5-разметка
- Смайлики вместо изображений для всех товаров и команды

## Технологии

- HTML5
- CSS3 (Flexbox, Grid, Media Queries, анимации)
- Git/GitHub
